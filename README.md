# Minora - E-Commerce Shoes Website

Minora is a modern front-end e-commerce website designed for showcasing and selling shoes online. This project focuses on clean UI, responsive design, and an engaging user experience.

---

## 🚀 Features

- Responsive design (mobile, tablet, desktop)
- Modern and clean UI/UX
- Product listing with images, names, and prices
- Category sections (Men, Women, etc.)
- Interactive buttons and hover effects
- Smooth navigation experience
- Stylish product cards layout

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
